
# GithubFinder-React

Search and get the most trending github repositories


## Dependencies

 - react-moment
 - redux
 - react-redux
 - redux-devtools-extension
 - express-async-handler
 - redux-thunk


## Installation

Install my-project with npm

```bash
  git clone https://github.com/TarokhDev2020/GithubFinder-React.git
  cd my-project
  npm install
```
